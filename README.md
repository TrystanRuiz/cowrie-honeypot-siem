This project covered SSH honeypot deployment and SIEM integration on Proxmox VE — Cowrie honeypot with Splunk log analysis, Hydra brute force simulation, and network hardening with UFW and Fail2Ban. It has been consolidated with the T-POT pipeline project since Cowrie is one of T-POT's core honeypot sensors.
